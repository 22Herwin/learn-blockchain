Single Block Mining

A block contains data and a nonce (number used once). Miners change the nonce until they find a hash that starts with four zeros (difficulty target).

Blockchain

Blocks are linked together using hashes. Each block references the hash of the previous block, creating an immutable chain.

ECC Digital Signature Verification

Elliptic Curve Cryptography (ECC) is used to sign and verify blockchain transactions. Generate a key pair, sign a message, and verify the signature.

Public and Private Keys can be edited for experimentation.
